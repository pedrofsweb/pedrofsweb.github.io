Web Journey

Over the past months, I've been enthusiastically exploring the 3 foundational technologies that power the Web: HTML, CSS and JavaScript.

After taking a few online courses to get started and reading some books to deepen my knowledge, I completed the "Front-End Web Developer" Professional Certificate. It is a program designed by the World Wide Web Consortium (W3C), creator of the Web standards.

Now that I think I have solid frontend roots, I'm moving forward on my JavaScript path by learning one of its many libraries — React. Once I feel comfortable enough with it, my goal is to build a client-side web application to showcase some of the skills I've acquired so far.
